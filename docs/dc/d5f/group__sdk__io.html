Перечисления

◆ MessageType

enum MessageType

Перечисление типов сообщения.

Содержит базовые типы сообщения. Перечисление можно расширить в классах-имплементациях этого интерфейса. Для этого необходимо описать новое перечисление, в котором первое значение будет равняться IMessage::MessageType + 1. Пример:

class CriticalMessage : public IMessage
{
public:
enum CriticalMessageType
{
OutOfMemory = IMessage::MessageType + 1,
SegmentationFault,
UnhandledException,
PermissionDenied
};
// ...
};
