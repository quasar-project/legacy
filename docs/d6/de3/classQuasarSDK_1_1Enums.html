Подробное описание

Класс-обёртка для общих перечислений SDK РЛС "Квазар".

Предоставляет общие перечисления для использования между различными классами внутри SDK, а также позволяет зарегистрировать их в мета-объектной системе для доступа из движка QML.

◆ UnixSignal

enum UnixSignal

Перечисление сигналов для процесса (https://ru.wikipedia.org/wiki/Сигнал_(Unix))

Элементы перечислений
SigHUP 

Уведомление: закрытие терминала.

SigINT 

Управление: сигнал прерывания (Ctrl-C) с терминала.

SigQUIT 

Управление: сигнал «Quit» с терминала (Ctrl-).

SigILL 

Уведомление: недопустимая операция процессора.

SigTRAP 

Отладка: ловушка трассировки или брейкпоинт.

SigABRT 

Управление: сигнал посылаемый функцией abort().

SigFPE 

Исключение: ошибочная арифметическая операция.

SigKILL 

Управление: безусловное завершение.

SigBUS 

Исключение: неправильное обращение в физическую память.

SigSEGV 

Исключение: нарушение при обращении в память.

SigSYS 

Исключение: неправильный системный вызов.

SigPIPE 

Уведомление: запись в разорванное соединение (пайп, сокет).

SigALRM 

Уведомление: сигнал истечения времени, заданного alarm().

SigTERM 

Управление: сигнал завершения (сигнал по умолчанию для утилиты kill).

SigUSR1 

Пользовательский: пользовательский сигнал № 1.

SigUSR2 

Пользовательский: пользовательский сигнал № 2.

SigCHLD 

Уведомление: дочерний процесс завершен или остановлен.

SigTSTP 

Пользовательский: сигнал остановки с терминала (Ctrl-Z).

SigURG 

Уведомление: на сокете получены срочные данные.

SigPOLL 

Уведомление: событие, отслеживаемое poll().

SigSTOP 

Управление: остановка выполнения процесса.

SigCONT 

Управление: продолжить выполнение ранее остановленного процесса.

SigTTIN 

Управление: попытка чтения с терминала фоновым процессом.

SigTTOU 

Управление: попытка записи на терминал фоновым процессом.

SigVTALRM 

Уведомление: истечение «виртуального таймера».

SigPROF 

Отладка: истечение таймера профилирования.

SigXCPU 

Исключение: процесс превысил лимит процессорного времени.

SigXFSZ 

Исключение: процесс превысил допустимый размер файла.
