Подробное описание

Класс, предоставляющий палитру контрастных цветов.

Для использования класс необходимо зарегистрировать в QML либо как тип, либо как синглтон. Предпочтительнее регистрировать класс как одиночный тип:

qmlRegisterType<Scenegraph::Utils::ColorProvider>("Scenegraph.Extras", 1, 0, "ColorProvider");

После этого необходимо импортировать тип в QML и создать объект:

import Scenegraph.Extras 1.0
Item {
ColorProvider {
id: provider
theme: ColorProvider.Light
behavior: ColorProvider.RandomCycle
}
// Пример получения цвета:
ListView {
model: 100
delegate: Rectangle {
width: 100
height: 100
color: provider.next()
}
}
}

◆ DistributionBehavior

enum DistributionBehavior

Перечисление поведения класса при вызове функции next().

Элементы перечислений
OrderedCycle 

При вызове функции next() будет возвращен следующий по порядку цвет из палитры. Когда палитра закончится, цвета снова будут браться из нее с начала.

RandomCycle 

При вызове функции next() будет возвращен случайный цвет из палитры. Он может повторяться.

OrderedTint 

При вызове функции next() будет возвращен следующий по порядку цвет из палитры. Когда палитра закончится, цвета в ней будут слегка изменены и снова будут браться из ее начала.

RandomTint 

При вызове функции next() будет возвращен случайный цвет из палитры. Если такой цвет был выдан ранее, он будет изменен.
